# Remove Instagram Distractions

A custom uBlock Origin filter list that removes Reels, suggested posts, right panel clutter, and other distracting elements from Instagram. No extra extension needed.

> By the maker of [Remove YouTube Distractions](https://github.com/Trazer09/Remove-YouTube-Distraction).

## Features

* No extra extension needed — works purely through uBlock Origin
* Remove Reels from the left navbar (both collapsed and expanded states)
* Hide suggested posts and "Suggested for you" section
* Remove the entire right panel (stories, messages, suggested accounts, footer)
* Hide in-video Reel interactions (like, comment, share, save, more)
* Clean up the UI for a distraction-free browsing experience

## Installation

1. Install [uBlock Origin](https://github.com/gorhill/uBlock).
2. Copy and paste the `filters.txt` content into the **My Filters** section of uBlock Origin and click **Apply Changes**.

## Notes

Some filters are commented out (`!` in front) — these are optional nuclear options like removing the entire left navbar or full Reel video overlay. Enable them manually if you want.

This list is actively maintained to keep up with Instagram's UI changes. If you find this repository useful, feel free to star ⭐ it to show your support.
